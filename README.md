# PyWazi
针对 ExHentai, JavBus, Danbooru, PicAcg 的一个数据采集和处理、操作模块。

目前版本是 1.0.0，已经完成了大部分内容。并完成了 Danbooru, JavBus, ExHentai（部分） 的测试，因为 PicAcg 我的移动网络不允许，挂梯子也不行，所以还未测试。年底之前（草）我会把开发文档（中英文双版）整出来，优化和功能增加会在以后完成。
